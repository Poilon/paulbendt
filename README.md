# PAUL BENDT — site portfolio (une page, statique, fait main)

Site **100 % statique, une seule page**. Que du HTML + un CSS. Aucun build, aucun framework.
Esprit carnet / fanzine : papier crème, encre bleue `#0069FF`, traits tremblés dessinés à la main.

## Voir le site
Double-clique sur `index.html`, ou un petit serveur local :

```bash
python3 -m http.server 8000   # puis http://localhost:8000
```

## Fichiers
- `index.html` — la page (hero, ce que je fais, liens, contact)
- `assets/style.css` — tout le design fait-main
- `assets/script.js` — apparitions au scroll + année du footer
- `assets/logo.png` — la mascotte (fond transparent)
- `assets/stamp.png` — le tampon « approved this site » (dans le coin)
- `assets/favicon.png` — icône d'onglet
- `paulbendt.JPG`, `Illustration_sans_titre.jpg` — images sources d'origine
